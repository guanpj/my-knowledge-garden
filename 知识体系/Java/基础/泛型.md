# 什么是泛型？

声明中具有一个或者多个 **类型参数(type parameter)** 的类或者接口，就是 **泛型类或者接口**。泛型类和接口统称为 **泛型(generic type)**。

String 就是对应于 **类型形参(formal type parameters)** 的 **类型实参(actual type parameter)**。
